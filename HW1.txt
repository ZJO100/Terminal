1) Посмотреть где я: pwd
2) Создать папку: mkdir D1
3) Зайти в папку: cd D1
4) Создать 3 папки: mkdir D11 D12 D13
5) Зайти в любую папку: cd D11
6) Создать 5 файлов (3 txt, 2 json): touch T1.txt T2.txt T3.txt J1.json J2.json
7) Создать 3 вложенных папки - mkdir -p F1/F2/F3
8) Вывести список содержимого папки: ls -la
9) + Открыть любой txt файл: vim T1.txt
10) + написать туда что-нибудь, любой текст:  I - режим редактирования, пишем

Можно cat >> T1.txt
11) + сохранить и выйти: esc — выйти из режима редактирования, :wq — сохранить изменения (для cat - Ctrl+C)
12) Выйти из папки на уровень выше: cd ..
13) переместить любые 2 файла, которые вы создали, в любую другую папку: mv D1/D11/T2.txt D1/D11/J2.json D2/
14) скопировать любые 2 файла, которые вы создали, в любую другую папку: cp D1/D11/T1.txt D1/D11/J1.json D3/
15) Найти файл по имени: find . -name "*.txt",  find . -iname "t1.txt", find . -iname "*1.*"
16) просмотреть содержимое в реальном времени: tail -f T3.txt (выйти из режима - Ctrl+C)
17) команда grep изучите как она работает: grep "дядя" T3.txt - поиск текста в файле, параметры:
-i - не чувствительный регистр,
-w - поиск по точному совпадению,
-n - выводит номер найденной строки,
-A2 - выводит 2 строки после найденной, 
-B2 - выводит 2 строки до найденной, 
-C2 - выводит 2 строки до и 2 после найденной, 
grep "roast-beef" ./* - поиск текста в тек. папке во всех файлах
-r - поиск не только в тек. папке, но и во вложенной
-l - покажет только файлы, где есть искомый текст,
-c - покажет сколько раз был найден текст,
grep -P "\d{3}-\d{2}-\d{2}" T3.txt- поиск регурярного вхождения, маска ищет цифры в формате 000-00-00,
если в конце ставим > F.txt - то найденные вхождения сохранятся в новый файл,
-o - выводит только то, что ищем
grep -owiPf pattern.txt T3.txt > F.txt - ищем вхождения, используя файл(pattern) с маской поиска (параметр f) и сохраняем найденное в файл
